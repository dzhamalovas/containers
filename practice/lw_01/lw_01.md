## Индивидуальное задание (Вариант 6 — Jupyter Notebook)

### Запуск контейнера
```bash
docker pull jupyter/base-notebook
docker run -d --name jupyter -p 8888:8888 jupyter/base-notebook
```
### Получение токена
```bash
docker logs jupyter
```
